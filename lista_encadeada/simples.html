Lista Encadeada simples

Uma lista encadeada simples é uma estrutura de dados linear que consiste em nós, onde cada nó contém dois campos principais:

  • Dado: Isso armazena o valor ou informação que você deseja armazenar na lista.
    • Ponteiro para o Próximo Nó: Isso aponta para o próximo nó na lista ou é definido como NULL quando o nó atual é o último na lista.

Exemplo de declaração de lista em C:
                
                    struct Node {
                        int data;          // Dado a ser armazenado
                        struct Node* next; // Ponteiro para o próximo nó
                    };

Nesta estrutura, data armazena o valor do elemento e next é um ponteiro para o próximo nó na lista.

Criando e Manipulando uma Lista Encadeada:
                
                    #include 
                        #include 
                        
                        struct Node {
                            int data;
                            struct Node* next;
                        };
                        int main() {
                            // Inicializa a lista vazia
                            struct Node* head = NULL;
                        
                            // Adiciona elementos à lista
                            head = (struct Node*)malloc(sizeof(struct Node));
                            head->data = 1;
                            head->next = NULL;
                        
                            // Adiciona mais elementos
                            struct Node* second = (struct Node*)malloc(sizeof(struct Node));
                            second->data = 2;
                            second->next = NULL;
                            head->next = second;
                        
                            // Adiciona mais elementos
                            struct Node* third = (struct Node*)malloc(sizeof(struct Node));
                            third->data = 3;
                            third->next = NULL;
                            second->next = third;
                        
                            // Percorre a lista e imprime os elementos
                            struct Node* current = head;
                            while (current != NULL) {
                                printf("%d ", current->data);
                                current = current->next;
                            }
                        
                            // Libera a memória alocada para os nós
                            current = head;
                            while (current != NULL) {
                                struct Node* temp = current;
                                current = current->next;
                                free(temp);
                            }
                        
                            return 0;
                        }

Neste exemplo, é criada uma lista encadeada com três elementos (1, 2 e 3) e, em seguida, percorre-se a lista para imprimir os elementos. Por fim, a memória alocada é liberada para os nós da lista.

Funcionamento por Trás da Linguagem

Por trás da linguagem C, a lista encadeada é implementada usando alocação dinâmica de memória para criar nós. Cada nó é alocado individualmente no heap (uma área de memória que pode ser gerenciada programaticamente) usando a função malloc. Os ponteiros next em cada nó são usados para conectar os nós, criando assim a estrutura encadeada.

É importante notar que, em C, você é responsável por liberar a memória alocada para os nós quando não precisar mais deles, para evitar vazamentos de memória. Isso é feito usando a função free. O exemplo acima demonstra como fazer isso após a utilização da lista encadeada.

Em resumo, uma lista encadeada simples é uma estrutura de dados flexível em C que permite armazenar e manipular uma coleção de elementos de forma dinâmica. Ela é construída por meio de nós que estão vinculados por ponteiros, e a alocação dinâmica de memória é usada para criar e gerenciar os nós da lista.
