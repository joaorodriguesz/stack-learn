Pilhas

Uma pilha é uma estrutura de dados linear que segue o princípio "último a entrar, primeiro a sair" (LIFO - Last In, First Out). Isso significa que o último elemento adicionado à pilha é o primeiro a ser removido. As operações básicas em uma pilha são a inserção de elementos (push) e a remoção de elementos (pop) na parte superior da pilha. Vou explicar o funcionamento de uma pilha e fornecer exemplos em C, além de discutir como ela é implementada nos bastidores.

Funcionamento de uma Pilha

Uma pilha pode ser comparada a uma pilha de pratos em que você adiciona ou remove pratos na parte superior da pilha. Aqui estão as operações principais:

  • Push (Inserção): Adicionar um elemento à pilha. O novo elemento se torna o topo da pilha.
  • Pop (Remoção): Remover o elemento do topo da pilha. O elemento é retirado da pilha.

Implementação de uma Pilha em C:
Em C, a implementação de uma pilha pode ser usando um array ou uma lista encadeada. 
                
                    #include 
                        #include 
                        
                        #define MAX_SIZE 100 // Tamanho máximo da pilha
                        
                        struct Stack {
                            int items[MAX_SIZE];
                            int top;
                        };
                        
                        // Função para inicializar a pilha
                        void initialize(struct Stack *stack) {
                            stack->top = -1;
                        }
                        
                        // Função para verificar se a pilha está vazia
                        int isEmpty(struct Stack *stack) {
                            return stack->top == -1;
                        }
                        
                        // Função para verificar se a pilha está cheia
                        int isFull(struct Stack *stack) {
                            return stack->top == MAX_SIZE - 1;
                        }
                        
                        // Função para adicionar um elemento à pilha (push)
                        void push(struct Stack *stack, int item) {
                            if (isFull(stack)) {
                                printf("Erro: Pilha cheia.\n");
                                return;
                            }
                            stack->items[++stack->top] = item;
                        }
                        
                        // Função para remover um elemento da pilha (pop)
                        int pop(struct Stack *stack) {
                            if (isEmpty(stack)) {
                                printf("Erro: Pilha vazia.\n");
                                return -1;
                            }
                            return stack->items[stack->top--];
                        }
                        
                        int main() {
                            struct Stack stack;
                            initialize(&stack);
                        
                            push(&stack, 1);
                            push(&stack, 2);
                            push(&stack, 3);
                        
                            printf("Elementos removidos da pilha: %d, %d\n", pop(&stack), pop(&stack));
                        
                            return 0;
                        }
Funcionamento por Trás da Linguagem:

Nos bastidores, uma pilha implementada usando um array mantém um índice (denominado top neste exemplo) que aponta para o topo da pilha. Quando você insere um elemento, ele é adicionado à posição indicada por top, e top é incrementado. Quando você remove um elemento, o elemento na posição top é removido, e top é decrementado.

A verificação de pilha vazia (isEmpty) é feita verificando se top é igual a -1, indicando que a pilha não contém elementos. A verificação de pilha cheia (isFull) é feita verificando se top é igual ao tamanho máximo da pilha menos 1.

A implementação de pilha usando lista encadeada segue um princípio semelhante, mas em vez de usar um array, os nós são encadeados uns aos outros, e o topo da pilha é representado pelo primeiro nó.

Em resumo, uma pilha é uma estrutura de dados que segue o princípio LIFO, com operações de push (inserção) e pop (remoção). É amplamente usada para rastrear informações temporárias em programas e tem aplicações em algoritmos como a recursão e o rastreamento de estados em processos.
