Arrays

Um array é uma estrutura de dados que permite armazenar um conjunto de elementos do mesmo tipo em uma sequência contígua de memória. Os elementos em um array são organizados em posições consecutivas de memória, e cada elemento é acessado por meio de um índice, que indica sua posição no array.

Principais características de um array:

  • Tipo de Dados Homogêneo: Todos os elementos de um array devem ser do mesmo tipo de dados. Por exemplo, um array de inteiros armazena apenas inteiros, e um array de caracteres armazena apenas caracteres.
  • Tamanho Fixo: O tamanho de um array é definido no momento da criação e não pode ser alterado posteriormente.
  • Índices: Os elementos em um array são acessados por meio de índices inteiros. O índice começa em 0 para o primeiro elemento e vai até tamanho do array - 1 para o último elemento.

Exemplo de declaração:
                
                    int numeros[5];
Representação em memória:
Índices 0 1 2 3 4
Memória 0x00000000 0x0040A8F3 0x00FFD450 0x7FFFFFFF 0xFFFF0000
Acessar um elemento específico:
                
                    int primeiro_elemento = numeros[0]; // Acessa o primeiro elemento (índice 0)
                    int terceiro_elemento = numeros[2]; // Acessa o terceiro elemento (índice 2)
Uso de um Array em C:

Os arrays são amplamente utilizados em C para armazenar e manipular coleções de dados. Você pode inicializar, atribuir, acessar e percorrer os elementos de um array de acordo com suas necessidades.

Exemplo de Uso de Array em C:
                
                    int numeros[5] = {1, 2, 3, 4, 5}; // Inicializa um array de inteiros 

                    for (int i = 0; i < 5; i++) {
                        printf("%d ", numeros[i]); // Imprime cada elemento do array
                    }

Neste exemplo, é inicializado um array numeros com valores de 1 a 5 e usa-se um loop for para percorrer e imprimir esses valores.
Fechamento:

Em resumo, um array é uma estrutura de dados fundamental em C que permite armazenar e acessar elementos em uma sequência contígua de memória. Embora a linguagem C trate os arrays de forma eficiente e direta, a compreensão de como eles são implementados por trás das cenas é importante para escrever código eficaz e otimizado.
