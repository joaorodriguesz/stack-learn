Filas

  • Enqueue (Inserção): Adicionar um elemento à fila. O novo elemento é inserido na parte de trás (ou final) da fila.
  • Dequeue (Remoção): Remover o elemento da frente da fila. O elemento é retirado da fila na ordem em que foi adicionado.

Implementação de uma Fila em C
Em C, a fila pode ser implementada usando um array ou uma lista encadeada. 
                
                    #include 
                    #include 
                    
                    #define MAX_SIZE 100 // Tamanho máximo da fila
                    
                    struct Queue {
                        int items[MAX_SIZE];
                        int front;
                        int rear;
                    };
                    
                    // Função para inicializar a fila
                    void initialize(struct Queue *queue) {
                        queue->front = -1;
                        queue->rear = -1;
                    }
                    
                    // Função para verificar se a fila está vazia
                    int isEmpty(struct Queue *queue) {
                        return (queue->front == -1 && queue->rear == -1);
                    }
                    
                    // Função para verificar se a fila está cheia
                    int isFull(struct Queue *queue) {
                        return (queue->rear == MAX_SIZE - 1);
                    }
                    
                    // Função para adicionar um elemento à fila (enqueue)
                    void enqueue(struct Queue *queue, int item) {
                        if (isFull(queue)) {
                            printf("Erro: Fila cheia.\n");
                            return;
                        }
                        if (isEmpty(queue)) {
                            queue->front = queue->rear = 0;
                        } else {
                            queue->rear++;
                        }
                        queue->items[queue->rear] = item;
                    }
                    
                    // Função para remover um elemento da fila (dequeue)
                    int dequeue(struct Queue *queue) {
                        if (isEmpty(queue)) {
                            printf("Erro: Fila vazia.\n");
                            return -1;
                        }
                        int item = queue->items[queue->front];
                        if (queue->front == queue->rear) {
                            queue->front = queue->rear = -1;
                        } else {
                            queue->front++;
                        }
                        return item;
                    }
                    
                    int main() {
                        struct Queue queue;
                        initialize(&queue);
                    
                        enqueue(&queue, 1);
                        enqueue(&queue, 2);
                        enqueue(&queue, 3);
                    
                        printf("Elementos removidos da fila: %d, %d\n", dequeue(&queue), dequeue(&queue));
                    
                        return 0;
                    }
Funcionamento por Trás da Linguagem:

Nos bastidores, uma fila implementada usando um array mantém dois índices, front e rear, que indicam a posição do elemento da frente e do elemento de trás da fila, respectivamente. Quando você insere um elemento, ele é adicionado à posição indicada por rear, e rear é incrementado. Quando você remove um elemento, o elemento na posição front é removido, e front é incrementado.

A verificação de fila vazia (isEmpty) é feita verificando se front e rear são iguais a -1, indicando que a fila não contém elementos. A verificação de fila cheia (isFull) é feita verificando se rear é igual ao tamanho máximo da fila menos 1.

A implementação de fila usando lista encadeada segue um princípio semelhante, mas em vez de usar um array, os nós são encadeados uns aos outros, e front e rear são ponteiros que apontam para o primeiro e último nó da fila, respectivamente.

Em resumo, uma fila é uma estrutura de dados que segue o princípio FIFO, com operações de enqueue (inserção) e dequeue (remoção). Ela é amplamente usada em programação para rastrear elementos em uma ordem específica, como em sistemas de gerenciamento de tarefas, impressão de documentos e muito mais.
